VR Testing

At the start of the second year, we started a new project. I worked on the project in a group of two. We chose sandbox/survival as the genre of the game, and also wanted it to be in VR. We ended up not finishing the game, but I did make a couple of cool features. For example I made a system to connect objects to other objects and the objects would connect to each other exactly how the player was holding them. I also made the tree/rock generation script for the game.

Mobile sandbox game

❮ ❯

A mobile game was the last school project of the first year. The game is a sandbox/destruction type game, where the player can play around with different blocks and destroy them. I made the game in a group of two, in which I made most of the main mechanics of the game and my friend made all of the graphics, like the textures and UI-elements and also a couple small scripts. For example the grid-system, the block connection script and the mechanics of the weapons were made by me.

3D Puzzle game

This project was a school project that I made with two other people. The idea of the game is that the player needs to control cameras and robots to solve a puzzle. I made the main mechanic of being able to control other cameras and robots. This project wasnt fully finished, since we ran out of time to make better lighting or texture objects and the level, but we did manage to make all the main mechanics of the game.
